Hướng Dẫn Chạy Project:
Bước 1: Download project từ github và open bằng Visual Studio Code.
Bước 2: Open terminal của project gõ cú pháp : npm start để chạy phần 
giao diện bằng reactjs.
Bước 3: Open terminal của folder api nằm trong folder src gõ cú pháp:
nodemon server để chạy phần backend bằng nodejs.
Bước 4:	Open Chrome, FireFox,... gõ localhost:3000 thì project sẽ hiện lên.
Bước 5 : Kết thúc.
